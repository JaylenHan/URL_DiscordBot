# 프로젝트 개요 (Project Brief)

## 한 줄 요약

채팅 속 URL의 악성 여부를 ML 모델 + WHOIS로 실시간 판별해 경고하는 Discord 봇.
KISIA AI보안 교육과정에서 개발, **장려상(전체 4위)** 수상.

## 문제 정의

- Discord에서 유포되는 악성 URL(피싱, 악성코드 배포, 위장 Nitro 코드 등)로 인한 사용자 피해
- 사용자가 실수로 악성 콘텐츠에 접근하지 않도록 **사전 차단·경고**가 필요

## 기획 의도

- **안전한 디스코드 환경 조성** — 악성 URL 같은 잠재 위협을 사전 차단
- **보안 문제에 대한 신속한 대응** — 최근 디스코드 보안 이슈를 분석해 실질적 방안 제공
- **디지털 안전성 강화** — 보안 인식을 높이고 안전한 온라인 활동을 장려

## 솔루션

URL 패턴을 분석하는 ML 모델과 whois/ipwhois 도메인 정보를 **결합**하여, 사용자가 접속하려는 URL의
악성 가능성을 실시간 분석하고 채널에 경고한다(이중 검증으로 정확도 향상).
